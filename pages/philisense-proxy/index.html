说明

虽然我们在开发过程中可以使用 hosts 文件构建虚拟主机域名，但在某些场景中，例如手机APP，就很难修改 hosts 文件，并且一旦服务器 IP 发生变化，我们需要修改多处 hosts 文件。

公共解析会造成私有项目在互联网中公开，所以本域名进行了防火墙拦截，不在 IP 白名单中的访问都会被拦截，可放心使用。

支持的域名

  • __.flxapp01.renfei.net
  • __.flxapp02.renfei.net
  • __.xz.renfei.net

使用方法

举个例子：

你在 flxapp01 服务器上配置 nginx，监听 80 端口，server_name 配置为 test.flxapp01.renfei.net

访问：http://test.flxapp01.renfei.net，即可访问你的项目。

如果你被防火墙拦截，那说明你的 IP 不在白名单中，需要联系 renfei 添加白名单：请先访问：ip.renfei.net，将你的 IP 地址告知我。。
