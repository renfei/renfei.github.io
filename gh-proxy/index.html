终端命令行 Terminal command line

支持终端命令行 git clone , wget , curl 等工具下载.

支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.

注意：不支持 SSH Key 方式 git clone 下载.

 

git clone

git clone https://github.renfei.net/https://github.com/renfei/repo-example

 

wget & curl

wget https://github.renfei.net/https://github.com/renfei/repo-example/archive/master.zip

wget https://github.renfei.net/https://raw.githubusercontent.com/renfei/repo-example/master/README.md

curl -O https://github.renfei.net/https://github.com/renfei/repo-example/archive/master.zip

curl -O https://github.renfei.net/https://raw.githubusercontent.com/renfei/repo-example/master/README.md
